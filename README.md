# 宠物领养系统
#### 介绍
宠物领养系统 技术栈springboot+vue+msql

#### 软件功能
管理员端:用户、公告、宠物品种、领养申请审核、领养中心管理等模块 用户端:宠物资讯、领养申请中心、公告管理等
